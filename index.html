Spins pulls art, tracklists, and full discographies straight from Spotify's catalog. Connect your account once — this only reads public catalog data.

Import legacy CSV data

This matches your old spreadsheet's Albums and EPs against Spotify's catalog and rebuilds each one here, track by track. Singles are matched best-effort by chronological release order per artist since your sheet has no titles for them — those get flagged as unverified so you can spot-check. Features aren't imported at all (no titles to match against). It has to look each one up individually, so it takes several minutes — stay on this tab while it runs.
